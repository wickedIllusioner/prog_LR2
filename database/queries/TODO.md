# TODO: SQL-запросы для БД

Здесь перечислены все запросы, которые необходимо реализовать.
Каждый запрос сохраняется в отдельном файле.

## Формат именования файлов

sql/queries/NN_query_краткое_описание.sql

Примеры:
- 01_query_high_severity_vuln.sql
- 02_query_server_assets.sql

Где NN — номер запроса от 01 до 30.

## Работа с запросами

1. Каждый выбирает запросы из списка
2. Создаёт ветку от актуальной main
3. Добавляет файл(ы) в папку `sql/queries/`
4. Создаёт Pull Request
5. После слияния — запрос считается выполненным
